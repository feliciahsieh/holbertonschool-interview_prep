This program deals with the [N-Queens puzzle](https://en.wikipedia.org/wiki/Eight_queens_puzzle), where the program will find N non-attacking Queens chess pieces that can be placed on an N Ã— N chessboard. The solution should use backtracking (recursion). The minimum size of the board is 4 x 4.
